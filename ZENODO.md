## Upload steps (manual)

1. Log in to [Zenodo](https://zenodo.org/) (GitHub account `sielerjm` is linked).
2. **New upload** → type: Dataset.
3. Title: `Sieler2026 DESeq2 checkpoint RDS — zebrafish stress contingency gut RNA-seq`
4. Description: DESeq2 fitted objects for module 04; companion to GitHub repo `zebrafish-stress-contingency-2026` and BioProject PRJNA1482558.
5. Upload `Sieler2026_dds_checkpoints.zip` from `publication_export/zenodo/` (generated by export script).
6. License: match OSU terms or CC0 for data artifact — confirm with Tom Sharpton if unsure.
7. Related identifiers: GitHub repo URL; BioProject PRJNA1482558.
8. Publish and copy the DOI into:
   - `README.md` and `DATA.md` (public repo)
   - Manuscript data availability (docx)
   - `CITATION.cff` optional `related_identifiers`
